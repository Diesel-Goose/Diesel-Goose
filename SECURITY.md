# SECURITY.md

**Security Policy for Greenhead Labs / Diesel-Goose Repo**  

**Versions Supported**  
- Current: main branch only.  

**Reporting a Vulnerability**  
- Email: security@greenheadlabs.com (or Telegram to @Greenhead_Labs prefixed “Chairman: SECURITY [details]”).  
- Response: Acknowledged in 24h; fixed in 72h if critical.  
- Bounty: Up to $10K for verified high-severity issues (crypto/AI/Web3 focus).  
- Do not disclose publicly until patched.  

**Best Practices Enforced**  
- Secrets scans via HEARTBEAT.md.  
- Signed commits required.  
- No hard-coded keys/wallets.  
- Report scans: Weekly via agents.
